RiSK

RiSK is a strategy game built using Java and JavaFX, featuring an intuitive GUI interface.

The game consists of a connected graph map representing a world map, where each node is a country and each edge represents adjacency between countries. Two or more players can play by placing armies on countries they own, from which they can attack adjacent countries to conquer them. The objective of the game is to conquer all countries on the map.

Game Rules: https://www.hasbro.com/common/instruct/risk.pdf


Features:

  • GUI interface built using JavaFX.
  • CPU mode with Novice, Balanced and Aggresive behaviour set.
  • Supports a tournament mode, which can include 2-5 players(or CPU).
  • Custom map creator and validator(only connected graphs are valid maps).

Technologies: Java, Junit, Javadoc, Checkstyle

Architecture: MVC, strategy pattern

Semantic Coupling Measurement

The tool measures semantic coupling between two Java Classes using Vector Space Model(VSM). Vector space model (VSM), also known as term vector model, is a commonly used model for representing text documents (and any objects, in general) as vectors of identifiers.

If two classes have high cosine similarity means the semantic coupling between the two classes is high. The tool returns the top k most similar Java classes for a given Java class.


Process:

  1. Use a vector of TF.IDF weights to represent each Java class, using preprocessing techniques(tokenization, remove stop words).
  2. Calculate the cosine similarity between the given Java class (i.e., the query ) and all the other Java classes (i.e., the collection of documents).
  3. Rank all the other Java classes based on the cosine similarity and return the top k java classes with the highest semantic coupling (i.e., cosine similarity).

Technologies: Java, Maven
